# 🚇 Sistema de Rotas do Metrô do Recife (Python)

Versão do projeto desenvolvida em Python, com o objetivo de calcular o tempo de viagem entre estações do metrô do Recife.

---

## 📌 Funcionalidades

- Seleção de estação de origem e destino
- Cálculo automático do tempo de trajeto
- Validação de entradas do usuário
- Interface via terminal

---

## 🛠️ Tecnologias utilizadas

- Python
- Lógica de programação

---

## ▶️ Como executar o projeto

### Pré-requisitos
- Python instalado (versão 3.x)

### Passos

```bash
git clone https://github.com/medeirosanderson/metro-recife-python.git
cd metro-recife-python
python main.py
```
